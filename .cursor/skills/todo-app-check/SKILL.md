# Todo App 動作確認 Skill

sample-app の Todo アプリが正しく動くか、以下の手順で確認する。

## チェックリスト

1. `sample-app/src/index.html` をブラウザで開く方法を案内
2. 以下を順に確認:
   - [ ] 新しい Todo を追加できる
   - [ ] チェックボックスで完了 / 未完了を切り替えられる
   - [ ] 削除ボタンで Todo を削除できる
   - [ ] 件数表示が正しい
   - [ ] ページをリロードしてもデータが残る（localStorage）
3. 問題があれば、該当ファイルと原因を報告
4. すべて OK なら「動作確認完了」と報告

## 制約

- コード変更は行わない（確認のみ）
- 問題発見時は修正案を提案するが、実装はユーザーの指示待ち
